🌐 Gender Perspective Translator  
A humorous mobile app that uses AI to “translate” what someone says into a stereotypical or exaggerated interpretation — strictly for comedy.  
Built as a side project, it combines Android development with generative AI for playful, over-the-top translations.  

📱 About the App  
The app takes everyday phrases like “I’m hungry” or “Do whatever you want” and runs them through an AI model that rephrases them into satirical “what they really mean” versions. It’s designed to make people laugh at exaggerated stereotypes — with filters to keep humor safe and lighthearted.  

💡 Features  
🤖 AI-Powered Humor – Generates unique, funny “translations” every time.  
🎭 Choose Gender – Translate either from man to woman or from woman to man  

⚙️ Tech Stack  
Kotlin  
Jetpack Compose  
OpenAI Api  
Kotlin Coroutines & Flow  
MVI
